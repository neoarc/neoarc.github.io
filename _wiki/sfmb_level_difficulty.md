# How difficult work?

- Level difficulty is automatically calculated based on the number of clears and the number of deaths (=Clear rate).

## Exact value

- 70 ~ 100 % : EASY
- 30 ~ 70 % : NORMAL
- 5 ~ 30 % : HARD
- 0 ~ 5 % : HELL

## Side note - Mario Maker

- 20 ~ 100 % : EASY
- 8 ~ 20 % : NORMAL
- 1 ~ 8 % : EXPERT
- 0 ~ 1 % : SUPER EXPERT
